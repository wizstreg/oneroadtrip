# OneRoadTrip — Mode d'emploi du kit

Bonjour,

Ce dossier contient tout ce qu'il faut pour ajouter OneRoadTrip sur votre site,
à côté de vos autres partenaires voyage. Tout est gratuit pour vos salariés.

Le lien à utiliser partout : https://www.oneroadtrip.com/

---

## Ce que contient le dossier

**Dossier "1_logos"**
Le logo OneRoadTrip en plusieurs versions.
- Fond transparent (s'adapte à n'importe quelle couleur de votre site).
- Version sur fond bleu, version tout doré, version blanche.
À utiliser dans votre liste de partenaires, en en-tête, etc.

**Dossier "2_bannieres"**
Des images toutes prêtes, avec le texte et un bouton, à poser sur votre site.
- banniere_300x250 : petit encart (colonne de droite, encadré).
- banniere_728x90 : bandeau fin (haut ou bas de page).
- banniere_1200x400 : grande bannière (en-tête de page).
- banniere_1080x1080 : format carré (réseaux sociaux, newsletter).

**Dossier "3_favicons"**
Petite icône du site (utile seulement si vous créez une page dédiée). 
Vous pouvez l'ignorer si vous ajoutez juste un lien et une bannière.

**Fichier "textes_pretsAColler"**
Des descriptions toutes faites (courte, moyenne, longue) à copier-coller
selon la place dont vous disposez.

---

## Comment faire, étape par étape

### Le plus simple : un logo + un lien
1. Prenez une image du dossier "1_logos".
2. Mettez-la sur votre page, comme vos autres partenaires.
3. Faites pointer le clic vers : https://www.oneroadtrip.com/

### Un peu plus visible : une bannière
1. Choisissez une bannière dans "2_bannieres" selon l'emplacement.
2. Posez-la sur votre page.
3. Faites pointer le clic vers : https://www.oneroadtrip.com/

---

## Le bout de code (si la personne qui gère le site le demande)

Pour afficher une bannière cliquable, ce code suffit :

    <a href="https://www.oneroadtrip.com/" target="_blank">
      <img src="banniere_1200x400.webp" alt="OneRoadTrip" style="max-width:100%;">
    </a>

Remplacez le nom du fichier par la bannière choisie.
